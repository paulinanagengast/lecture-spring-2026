## Overview:

This is a length converter for converting measurements between Metric and Imperial systems. It can convert Meter, Foot and Inch. 
Procedure:
1. Input of the number to be converted.
2. Input of the unit to be converted.
3. Input of the desired unit.

Dies ist ein Längenkonverter zum Umrechnen von Messungen zwischen dem metrischen und dem imperialen System. Er kann Meter, Fuß und Inch umrechnen. 
Vorrgehensweiße:
1. Eingabe der zu konvertierten Zahl.
2. Eingabe der zu konvertierenden Einheit.
3. Eingabe der gewünschten Einheit.

AI was used for programming the correct python syntax.

KI wurde verwendet, um die korrekte Python-Syntax zu programmieren.
